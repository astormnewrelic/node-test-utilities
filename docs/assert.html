(static) exactSegments(parent, expected)

Asserts that the segments are exactly structured as expected.

Ordering of children as well as the amount of children are strictly checked as part of this assertion. In addition, the name must be provided in full.

Parameters:
Name Type Description
parent TraceSegment

The segment to check the children of.
expected Array.<Object>

The expected children.

Properties
Name Type Attributes Default Description
name string

The full and exact name of the segment.
children Array.<Object>

An array of all expected children. This should match the same structure as the expected parameter.
exact bool <optional>
 true

Indicates if the children should be matched exactly. If exact: false is given then from these children on, until exact: true is specified, segments will be compared loosely.

Asserts that the segments are roughly structured as expected.

Ordering of children as well as amount of children are not considered as part of this assertion. In addition, the name is only loosely matched; the segment name need only contain the provided string.

 false

Indicates if the children should be matched exactly. If exact: true is given then from these children on, until exact: false is specified, segments will be compared strictly.
